Create src/commands/questions.js with three command functions:

1. cmdQuestionsAudit(cwd, raw):
   - Scan all .md files in workflows/ directory
   - Detect inline questions: question([, Use question:, Ask: patterns
   - Detect questionTemplate() usage
   - Report taxonomy compliance percentage
   - Default: human-readable Markdown output
   - --json flag: machine-readable JSON for CI

   - Read src/lib/questions.js OPTION_TEMPLATES
   - List all templates with: id, type (TAXONOMY), option count, usage count
   - Default: human-readable Markdown output
   - --json flag: machine-readable JSON

3. cmdQuestionsValidate(cwd, raw):
   - Read OPTION_TEMPLATES
   - Validate each template: MIN 3 options, MAX 5 options, escape hatch present
   - Check formatting parity (same length/grammar/detail across options)
   - Default: warn-only (report issues, don't fail)
   - --json flag: machine-readable JSON

Follow existing command patterns from workflow.js:
- Use lazy loading
- Output to stderr for human-readable, output() for JSON
- Include --json flag handling via raw parameter
