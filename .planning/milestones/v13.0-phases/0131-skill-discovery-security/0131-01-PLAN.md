Create `src/commands/skills.js` following the lessons.js / agent.js module pattern:

**Imports:** `fs`, `path`, `{ output, error, debugLog }` from `../lib/output`.

**Security Scanner — `SECURITY_PATTERNS` array (41 patterns in 7+ categories):**

Each pattern: `{ id, category, severity, pattern (RegExp), description }`. Severity: `'dangerous'` or `'warn'`.

Categories and patterns (41 total):
1. **Code Execution (dangerous):** `eval(`, `new Function(`, `vm.runInContext`, `vm.createContext`, `vm.Script`, `child_process`, `execSync`, `spawnSync`, `exec(`, `spawn(`
2. **Network Access (warn):** `fetch(`, `http.request`, `https.request`, `XMLHttpRequest`, `WebSocket`, `net.connect`, `dgram`
3. **Filesystem Operations (warn):** `fs.writeFile`, `fs.unlink`, `fs.rmdir`, `fs.rm(`, `fs.rename`, `fs.copyFile`, `fs.appendFile`
4. **Process/System (dangerous):** `process.exit`, `process.kill`, `process.env` (warn), `os.exec`, `require('os')` (warn)
5. **Crypto Mining (dangerous):** `stratum+tcp`, `cryptonight`, `coinhive`, `minergate`
6. **Data Exfiltration (dangerous):** `Buffer.from.*base64`, `btoa(`, `encodeURIComponent.*http`, `document.cookie`, `localStorage`
7. **Prompt Injection (dangerous):** `<system>`, `[INST]`, `</system>`, `IGNORE PREVIOUS`, `ignore all prior`, `you are now`, `new instructions:`

Total: ~41 patterns covering the 7 categories above.

**`scanSkillFiles(skillDir)` function:**
- Reads all files in `skillDir` recursively
- For each file, runs all 41 patterns against content (line-by-line for line numbers)
- Returns `{ verdict: 'clean'|'warn'|'dangerous', findings: [{ file, line, pattern_id, category, severity, match }], summary: { dangerous: N, warn: N, clean: N, total_patterns: 41 } }`
- Verdict logic: any dangerous finding → `'dangerous'`; any warn finding → `'warn'`; else `'clean'`

**`formatScanResults(scanResult, verbose)` function:**
- Groups findings severity-first: dangerous (red), then warn (yellow)
- Shows progress checklist with marks per category: `"✓ eval patterns  ✓ network access  ✗ filesystem ops (2 warnings)"`
- Clean scans: one-liner `"Security scan passed (41 patterns, 0 findings)"`
- Count summary at end: `"2 dangerous, 3 warnings, 36 clean"`
- If `verbose`: show matched code snippets per finding
- If dangerous: append `"Install BLOCKED — dangerous patterns found"` message

**`cmdSkillsList(cwd, options, raw)` function:**
- Read `.agents/skills/` directory (relative to `cwd`)
- For each subdirectory containing `SKILL.md`: extract name (dir name), extract description from SKILL.md first `## Purpose` or `description:` frontmatter field
- Run `scanSkillFiles` per skill to get scan status
- Format: aligned list with name + description + scan status (`✓ clean` / `⚠ 3 warnings`)
- Empty state: `"No skills installed."` (no hints per CONTEXT.md)
- JSON output: `{ skills: [{ name, description, path, scan_status }] }`
