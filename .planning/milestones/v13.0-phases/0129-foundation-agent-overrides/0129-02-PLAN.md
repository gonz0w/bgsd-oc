**In src/commands/agent.js — add `cmdAgentOverride(cwd, raw, args)`:**

1. Extract `<name>` from args[0]. If missing, error with usage hint: `'Usage: agent override <name>'`
2. Strip `.md` extension from name if provided (normalize to bare name)
3. Resolve global agent path: `resolveBgsdPaths().agentsDir` → `path.join(agentsDir, name + '.md')`
4. If global agent file does not exist:
   - Scan all global agents to find closest match using simple Levenshtein distance or substring matching
   - Error: `'Agent "${name}" not found. Did you mean "${closest}"?'`
   - If no close match: `'Agent "${name}" not found. Run "agent list" to see available agents.'`
5. Resolve local agents dir: `path.join(cwd, '.opencode', 'agents')`
6. Resolve local agent path: `path.join(localDir, name + '.md')`
7. If local override already exists:
   - Error: `'Agent "${name}" already has a local override at ${localPath}. Use "agent diff ${name}" to view changes or "agent sync ${name}" to update.'`
8. Read global agent file content
9. Call `validateAgentFrontmatter(content)` (from Plan 01):
   - If invalid: hard error with validation message — do NOT write the file
   - Note: Research found that current global agents don't have a `name:` field. The override command should inject `name: <agent-name>` into the frontmatter if it's missing, then re-validate. This ensures the local copy always has the required field.
   - To inject: parse the frontmatter section, add `name: <agent-name>` as the first field after `---`, reconstruct the file
10. Create `.opencode/agents/` directory if it doesn't exist: `fs.mkdirSync(localDir, { recursive: true })`
11. Write the (potentially name-injected) content to the local path
12. Output: just the file path created (per CONTEXT.md — no hints or suggestions)
   - Raw mode: `{ created: localPath, agent: name }`
   - Human-readable: just print the path

**Fuzzy matching helper — `findClosestAgent(name, agentNames)`:**
- Simple approach: check if any agent name contains the input as a substring, or shares a common prefix of length >= 4
- If multiple matches, return the shortest (most specific)
- If no match, return null

Export `cmdAgentOverride` from module.exports.

10. If differences exist:
   - Human-readable: print the diff string directly (it's already formatted)
